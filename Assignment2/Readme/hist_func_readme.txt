- Purpose of this code
 This code is written to generate PDF for single channel image and for color image, CDF for single channel image and for color image.
 
- How to run this code
 Through the visual studio.

- How to adjust parameters (if any)
 Input Mat of single channel image of both PDF, CDF is gray-scale image.
 Input Mat of color image of both PDF, CDF is color image. 

- How to define default parameters
 The parameters of all functions are input image(gray scale or color image)
