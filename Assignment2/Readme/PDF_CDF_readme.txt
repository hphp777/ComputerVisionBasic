- Purpose of this code
The purpose of this code is to calculate the number of each level of intensity of the input images. 
PDF: count the number of each level of intensity respectively
CDF: count the number of each level of intensity accumulatively

- How to run this code
Through the visual studio, load the PDF_CDF.cpp and run.

- How to define default parameters
