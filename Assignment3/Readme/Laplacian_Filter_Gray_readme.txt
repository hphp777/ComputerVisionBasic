- Purpose of this code
 Laplacian Filter is usually used to detect edges in the input image. This is one of the high-pass filter

- How to run this code
 Through Visual Studio, load 'Laplacian_Filter_Gray.cpp' with the input image 'lena.jpg'.

- How to adjust parameters (if any)
 Mat LaplacianFilter(const Mat input);
 const Mat input: User of this code only have to give input gray scale image.
